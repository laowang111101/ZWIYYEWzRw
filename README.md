# 加油站管理系统 java432

## 项目概述

本项目《加油站管理系统》采用前后端分离的架构设计，前端使用 Vue.js，后端采用 SpringBoot，数据库为 MySQL。系统主要包括论坛模块、加油站模块、汽油模块、加油模块以及部门角色菜单模块，适用于需要进行加油站管理的场合。

## 技术栈

- 前端：Vue.js
- 后端：SpringBoot
- 数据库：MySQL

## 功能模块

- 论坛模块
- 加油站模块
- 汽油模块
- 加油模块
- 部门角色菜单模块

## 系统角色

- 系统管理员
- 加油站工作人员
- 部门角色

## 运行环境

- 前端：现代浏览器
- 后端：JDK 1.8 或以上版本
- 数据库：MySQL 5.7 或以上版本

## 部署步骤

1. 搭建后端运行环境（JDK，MySQL）
2. 创建数据库，导入数据
3. 部署后端服务（SpringBoot）
4. 部署前端项目（Vue.js）
